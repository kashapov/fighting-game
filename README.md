# Contra Fight
Простая файтинг игра с задачами

## Варианты задач: 
1) решить арифметическую задачу
2) перевод с английского на русский
3) перевод с русского на английский
4) написать слово (на русском или английском) по картинке
5) отгадать загадку
6) логическая задача
